## How It Works
- The game shows a sequence of colors
- Repeat the pattern by clicking the buttons
- Each round adds a new color
- One mistake resets the game

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Play
1. Open the game in your browser
2. Start the game
3. Memorize the color sequence
4. Repeat the pattern to advance in levels
